# 圖表目錄

`figures/` 用來放論文需要引用的圖表、流程圖、架構圖與實驗結果圖。

## 命名建議

```text
fig_01_research_framework.png
fig_02_system_architecture.png
fig_03_experiment_result.pdf
table_01_dataset_summary.csv
```

## 每張圖表應記錄

- 圖表名稱：
- 對應論文章節：
- 產生方式：
- 對應程式：
- 對應資料：
- 是否可重現：
- 圖說草稿：

## 原則

- 論文中的圖表應能追溯到資料、程式或手動繪製來源。
- 統計圖與實驗圖應盡量由程式產生。
- 正式論文使用前，需確認解析度、標籤、字體與引用格式。
