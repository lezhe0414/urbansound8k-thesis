# AI Agent 使用揭露草稿

這份文件是預備用草稿。是否需要放入正式論文，應以學校規範與教授要求為準。

## 目前已知

- 指導教授已同意使用 AI Agent 協助本論文專案。
- AI Agent 用於協助專案管理、論文規劃、程式開發流程、文件模板與可重複研究規範。
- 正式研究內容、文獻引用、實驗結果與論文主張仍需由使用者確認與負責。

## 可能揭露文字

以下文字僅供草稿使用，正式版本需依教授與學校要求修改：

```text
本研究過程中，作者在指導教授同意下使用 AI Agent 作為輔助工具。AI Agent 主要協助研究流程管理、論文章節規劃、程式碼開發輔助、文件整理與文字潤飾。所有研究問題、資料使用、方法選擇、實驗結果、文獻引用與最終論文內容，均由作者檢查、判斷並負責。
```

- 是否需要在正式論文中揭露 AI Agent 使用？

- 是否需列出使用的 AI 工具名稱與版本？
