# 給教授的更新與詢問模板

這份文件用來快速整理要傳給教授的更新、詢問或會後確認。實際送出前，請依教授習慣、學校語氣與具體情境調整。

## 使用時機

- 想請教授確認論文方向。
- 想確認程式需要做到什麼程度。
- 想詢問資料來源、文獻或評估方式。
- 會議前想先寄出討論重點。
- 會議後想整理共識與下一步。

## 最短版

```text
老師您好，我目前想確認論文方向與下一步工作。

我的暫定方向是：

我預計需要撰寫的程式是：

我想請老師確認：
1. 這個方向是否合適？
2. 程式需要做到什麼程度？
3. 下一次會議前我應優先完成什麼？

謝謝老師。
```

## 正式版

```text
老師您好：

我目前正在整理論文計畫與程式實作方向，想先向您確認目前方向是否合適。

一、暫定論文方向

二、目前理解的研究問題

三、預計程式實作內容

四、目前資料來源或資料需求

五、想請老師確認的問題
1. 此研究方向是否需要再縮小或調整？
2. 程式實作需要達到資料分析、模型比較、系統原型或其他程度？
3. 是否有建議優先閱讀的文獻或比較方法？
4. 下一次會議前，您建議我優先完成哪一項產出？

謝謝老師。
```

## 會議前確認版

```text
老師您好，為了讓下次討論更聚焦，我先整理目前想和您確認的事項：

1. 論文題目或研究問題是否合適。
2. 程式實作範圍與最低完成標準。
3. 可使用的資料來源與資料限制。
4. 需要比較的 baseline 或相關方法。
5. 下一次會議前應完成的工作。

若老師方便，也想請您建議我優先閱讀的文獻或參考方向。

謝謝老師。
```

## 會後整理版

```text
老師您好，我整理本次討論後的理解如下，想請您確認是否正確：

1. 論文方向：
2. 研究問題：
3. 程式實作：
4. 資料來源：
5. 評估方式：
6. 下一次會議前要完成：

若以上理解有需要修正的地方，再麻煩老師提醒。謝謝老師。
```

## AI Agent 協作說明版

若需要向教授說明 AI Agent 如何協助，可使用以下草稿：

```text
老師您好，我會在您同意的範圍內使用 AI Agent 協助論文專案管理、程式碼撰寫輔助、文獻整理、實驗紀錄與文字潤飾。研究方向、資料使用、方法選擇、實驗結果與最終內容仍會由我確認並負責。若學校或老師需要，我也會保留 AI 使用紀錄。
```
