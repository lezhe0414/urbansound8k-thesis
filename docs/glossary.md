# 術語表

這份文件用來統一論文中的專有名詞、英文縮寫、中文翻譯與使用方式。當研究方向確定後，請逐步補上領域術語。

## 使用方式

- 新增重要術語時，先記在這裡。
- 同一術語在全文應使用一致譯名。
- 英文縮寫第一次出現時，應寫出完整名稱。
- 若教授或領域慣例有指定用法，以指定用法為準。

## 縮寫規則

- 第一次出現：中文譯名（English Full Name, ABBR）
- 後續出現：中文譯名或縮寫，依全文一致性決定。

範例：

```text
大型語言模型（Large Language Model, LLM）
```

## 待補

- 論文研究領域的核心術語。
- 程式或模型相關術語。
- 資料集或評估指標名稱。
- 教授指定用語。
