# 任務 Inbox

這份文件用來暫存還沒整理到正式里程碑、論文章節或程式任務中的事項。當你想到任何要做的事，可以先丟到這裡，再由 AI Agent 協助分類。

## 使用方式

1. 先把想到的任務寫到「未分類」。
2. 每次整理時，把任務移到適合的分類。
3. 已經排進 `docs/milestones.md` 的任務，可以標記為已整理。
4. 完成的重要任務應同步更新 `docs/research_log.md`。

## 未分類

- [ ] 補上論文題目或大方向。
- [ ] 補上教授最近要求完成的事項。
- [ ] 說明程式需要完成什麼功能。

## 寫作

- [ ] 補完 `docs/thesis_plan.md`。
- [ ] 建立第一版 `docs/thesis_outline.md`。

## 文獻

- [ ] 列出第一批核心文獻。
- [ ] 將每篇文獻整理到 `references/literature_notes.md`。

## 程式

- [ ] 決定程式語言與執行環境。
- [ ] 決定第一個原型要放在 `src/` 或 `notebooks/`。

## 資料

- [ ] 確認資料來源。
- [ ] 確認資料是否可公開或需要匿名化。

## 實驗

- [ ] 決定第一個可驗證的實驗目標。
- [ ] 決定評估指標或比較基準。

## 教授回饋

- [ ] 把已知教授要求補進 `docs/meeting_notes.md`。

## 已整理

暫無。
