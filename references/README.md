# 文獻與引用管理

`references/` 用來放文獻筆記、引用追蹤、BibTeX 或其他引用管理輸出。

## 建議文件

- `literature_notes.md`：逐篇文獻閱讀筆記。
- `citation_tracker.md`：追蹤哪些論文章節需要引用哪些文獻。
- `references.bib`：若使用 BibTeX，可放正式引用資料。

## 工作流程

1. 先把重要文獻整理到 `literature_notes.md`。
2. 如果某段論文需要引用但還沒找到來源，先標示 `[citation needed]`。
3. 將引用需求整理到 `citation_tracker.md`。
4. 確認學校或教授要求的引用格式。
5. 定稿前檢查正文引用與參考文獻清單是否一致。

## 注意

- 不要只留下 PDF，應至少有摘要與可引用重點。
- 引用文獻時要能說明該文獻支撐哪個論點。
- 若引用 AI 工具協助產出的文字或分析，需依學校規範處理。
